On Windows, you have several built-in ways to select part of the screen and capture it as an image:

🔹 1. Snipping Tool (Windows 10 & 11)

  1. Press Windows + Shift + S.

  2. Your screen will dim, and a small toolbar will appear at the top.

    • Rectangular Snip → draw a rectangle around what you want.

    • Freeform Snip → draw a custom shape.

    • Window Snip → capture a specific window.

    • Full-screen Snip → capture the entire screen.

  3. Once you select the area, it’s copied to your clipboard.

    • Paste with Ctrl + V in apps like Word, Paint, or any chat.

    • Or click the notification that pops up to save it as a file.

🔹 2. Snipping Tool App (more options)

  1. Press Windows key, type Snipping Tool, and open it.

  2. Click New → select your area.

  3. You can edit, annotate, and save the screenshot.

🔹 3. Print Screen with Crop (less direct)

  1. Press Print Screen (PrtScn) → captures the whole screen.

  2. Open Paint (or any editor) → press Ctrl + V.

  3. Crop the part you want and save it.

👉 Quickest method: Windows + Shift + S, then paste.

Here’s how you can actually execute this setup (either manually or by letting Cursor handle it):

🔧 Step 1 — Install TailwindCSS

Inside your React frontend folder (usually client/ in a MERN app):

bash
npm install -D tailwindcss postcss autoprefixer
npx tailwindcss init -p

Update tailwind.config.js like this:

js
/** @type {import('tailwindcss').Config} */
module.exports = {
  content: [
    "./src/**/*.{js,jsx,ts,tsx}"
  ],
  theme: {
    extend: {},
  },
  plugins: [],
}

In your src/index.css (or globals.css), add:

css
@tailwind base;
@tailwind components;
@tailwind utilities;

🔧 Step 2 — Install shadcn/ui

Run inside your frontend project:

bash
npx shadcn-ui@latest init

This creates a components.json file.
Then generate the components you need:

bash
npx shadcn-ui@latest add button card

This will create src/components/ui/button.jsx, src/components/ui/card.jsx, etc.

🔧 Step 3 — Install Extra Dependencies

bash
npm install lucide-react react-router-dom

🔧 Step 4 — Create Project Structure

Inside src/, add:

bash
components/ui/   ← (shadcn components live here)
layout/AppLayout.jsx

Paste the AppLayout.jsx code I gave you earlier (Header + Hero + Main + Footer).
